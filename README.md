## Phase 1 — A1: Foundation (Months 1–4)

### Goal
Introduce yourself, count, name objects, basic greetings, buy things, ask for help.

### Grammar Priorities
- German alphabet & pronunciation rules
- Personal pronouns: ich, du, er/sie/es, wir, ihr, Sie/sie
- Present tense (Präsens): sein, haben, regular verbs
- Nouns + articles (der/die/das) — the big three genders
- Negation: nicht / kein
- Basic sentence order: Subject–Verb–Object
- Numbers 1–100, dates, times

### Core Vocabulary (600 words target)
See `vocab/a1-wordlist.md` for the full list.

Key clusters:
- Greetings & introductions
- Family members
- Food & drink
- Colors, numbers, days, months
- Common verbs: gehen, kommen, essen, trinken, wohnen, arbeiten

### Hindi/Marathi Speaker Advantage
German and Hindi share **Indo-European roots**. Cognates to notice:
- Mutter (mother) ← माता
- Bruder (brother) ← भ्राता  
- Name (name) ← नाम
- Drei (three) ← तीन (notice the root)
- Neu (new) ← नव
- Wasser (water) ← वर (root)

## Phase 2 — A2: Survival German (Months 5–8)

### Goal
Handle everyday situations: shopping, appointments, directions, simple workplace talk.

### Grammar Priorities
- Past tense: Perfekt (most spoken German uses this, not Präteritum)
- Modal verbs: können, müssen, wollen, dürfen, sollen, mögen
- Accusative case (den/einen)
- Dative case (dem/einem) — for prepositions: mit, zu, bei, nach, von
- Separable verbs: aufmachen, anrufen, einladen
- Comparative adjectives: größer, kleiner, besser

## Phase 3 — B1: Conversational (Months 9–14)

### Goal
Hold real conversations, understand main points of clear speech, write simple connected text.

### Grammar Priorities
- Präteritum (simple past) — for written/formal German
- Konjunktiv II — würde, wäre, hätte (polite requests, hypotheticals)
- Two-way prepositions (Wechselpräpositionen): in, an, auf, über, unter...
- Genitive case
- Relative clauses: "Das ist der Mann, der..."
- Subordinating conjunctions: weil, dass, obwohl, wenn, als

### Vocabulary Target: 2,000 words

### Daily Schedule (aim for 45–60 min)

## Phase 4 — B2: Work-Ready (Months 15–22)

### Goal
Understand complex texts, communicate fluently with native speakers, discuss professional topics.

This is the **most important level for working in Germany** — most employers and visa categories require B2 minimum.

### Grammar Priorities
- Passiv constructions: wird gemacht / wurde gemacht / ist gemacht worden
- Extended participial phrases (complex written German)
- Konjunktiv I (reported speech in news/formal writing)
- Nominalization patterns
- Complex sentence structures

### Vocabulary Target: 4,000 words + work-domain vocabulary

## Hindi/Marathi Speaker — Special Notes

### Your Advantages
1. **Gender sensitivity** — Hindi has 2 genders (masculine/feminine), German has 3. You already think in grammatical gender.
2. **Case system** — Hindi has cases (kaaraka). German's 4-case system (Nominativ, Akkusativ, Dativ, Genitiv) will feel logical, not foreign.
3. **Postpositions → Prepositions** — Hindi uses postpositions; German uses prepositions. Flip the position, same concept.
4. **Verb conjugation** — Hindi conjugates verbs by person; so does German. The logic is familiar.
5. **Script** — You already know a non-Latin script. Learning the Latin alphabet for German phonetics is straightforward.

### Your Challenges
1. **Articles (der/die/das)** — English has no gender articles; Hindi has only 2. German has 3 with 4-case declension. **Learn the article WITH every noun from day 1.** Never memorize a noun alone.
2. **Verb-at-end rule** — Subordinate clauses push the verb to the end: _Ich weiß, dass er **kommt**._ This is actually similar to Hindi SOV structure!
3. **Compound words** — German strings nouns: _Donaudampfschifffahrtsgesellschaftskapitän_. Learn to break them apart.
4. **Umlauts** — ä, ö, ü are new sounds. Practice: ä = between 'a' and 'e', ö = round lips and say 'e', ü = round lips and say 'i'.

### Hindi–German Cognate Boost (memorize these first)
| German | Hindi | Meaning |
|--------|-------|---------|
| Mutter | माँ / माता | mother |
| Vater | पिता (root) | father |
| Bruder | भ्राता (root) | brother |
| Name | नाम | name |
| Neu | नव | new |
| Drei | तीन (root: tri) | three |
| Sechs | छह (root: shash) | six |
| Sieben | सात (root: sapta) | seven |
| Nacht | रात (root: nakt) | night |
| Gott | देवता (root: gott) | God |

## Quick Reference — German Pronunciation for Hindi Speakers

| German Sound | How to Pronounce | Hindi Approximation |
|-------------|-----------------|-------------------|
| ch (nach) | like clearing throat softly | ख़ (kh) |
| ch (ich) | softer, front-of-mouth hiss | श + ख combined |
| ü | round lips, say "i" | No Hindi equivalent — practice! |
| ö | round lips, say "e" | No Hindi equivalent — practice! |
| ä | open "e" sound | ऐ (ai) |
| w | like English "v" | व |
| v | like English "f" | फ़ |
| z | like "ts" in "cats" | त्स |
| ß (eszett) | long "ss" sound | स्स |
| r | uvular (back of throat) | guttural — not Hindi 'र' |
